Lyst

Lyst is a To-Do List Application for android based on Google's Material Design Standards.

Themes

Currently, the only available theme is Red. Well, but as you know this is an open source Project, so just fork it up and use your imagination and fire up a new theme, then post up a pull request. Every developmental pull request is appreciated.

Why Would You want this?

Well, this was made from scratch for personal use as well as to learn android coding, and hey, there's no harm in learning new stuff and sharing it right?

Importing the project

Download the sources through any git or download the source zip from here

Open Android Studio and Just Import the project (You might want to select "Open Existing Android Studio project" since gradle is already included)
